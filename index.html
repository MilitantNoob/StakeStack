Android Installation:

1. Look for the Install button in your browser's address bar

2. Tap the Install button to add to home screen

3. Confirm installation when prompted

iOS Installation:

1. Tap the Share button (square with arrow) at the bottom

2. Select Add to Home Screen

3. Tap Add to confirm

Browser Limitations Notice:

Safari on iOS has limitations that may affect popup functionality. For full features including social logins and external wallet connections, we recommend using Chrome or Firefox browsers.

iOS users can also use the Chrome app from the App Store to access all features.
